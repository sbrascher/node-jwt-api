# node-jwt-api

<b>Required</b>:
- [NodeJs](https://nodejs.org)
  - Express
  
- [MongoDB](https://www.mongodb.com/download-center/community)

Steps:

  $> git clone https://github.com/avmesquita/node-jwt-api.git
  
  $> cd node-jwt-api
  
  $> cd src
  
  $> npm install --save express
  
  $> node app -port 2727
